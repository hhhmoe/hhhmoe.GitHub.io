高考加油！！！

2020-04-01 3 min read

今天是2020年4月1日，距高考97天（话说距高考还有69天的场景仿佛就像在前天一样真实😂）
算是见证了历史吧 (:3[▓▓]快醒醒开学了
前几天考了一次模拟，题很水但我考的一点也不好，很伤心因为我很有可能这辈子再也碰不到这么简单的模拟题了，想想自己浪费了多么好的一次机会觉得难过，，但这不是重点，居然是我最擅长的数学和物理拉了太多得分，一方面是自己前些时间确实不够自觉，另一方面有轻视了自己平日所擅长的学科，有拆东墙补西墙的狼狈
= = = = = = == = = == = = == = = =郑重的分割线= = = == = = == = = == = = == = =

远方很远 步履不停
对着晚霞叹息不如对着朝阳奔跑，感谢上帝又给我30天的机会，那么目标自然也要放的更远一些。

先列个小目标

看起来这么多分很难涨，但实际上在没有碰到660分的第一个势垒，前面的分数可以在一个月内很快的增长，真正难的是先前的基础和最后的拔高
举个栗子🍭

  • 语文作文在上到49分后冲击50分是一个坎，一分需要很大的积累才能冲破，而且到大神级别（像LBX姐姐）一样的高考55+级别更是需要无数的时间（我是够呛了😌）
  • 可以通过表格发现，拉分数最多的往往是数学和语文，数学可以通过刷题短期内提高大量的分数，这也是近期我的目标（因为前期数学咕咕咕了才导致数学考这么差☹️）

自我要求😝

  • 在疫情在家期间，班里盛行搜题和私下对答案，一定要控制自己，只允许学习为目的的搜题和查资料，在考试期间绝对不允许用手机查答案
  • 严格限时，数学考试这次班级平均分都快到135了，很大原因是延时30min的因素，但是一定要拒绝诱惑
  • 注意心态的把控，模拟紧张感
  • 争取每天在这里更新一个简短的打卡总结反思自己的过程
  • 争取每一次机会，不仅仅在关键时刻展现实力，否则大概率在关键时刻翻车😐
